INTRODUÇÃO: O Elemento HTML "OL" (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

Exemplo:

  1. Exemplo
  2. Exemplo
  3. Exemplo
  4. Exemplo

Diferença entre os elemetos "ol e ul":

Ambos os elementos "ol" e "ul" representam uma lista de itens. São diferentes porque, com o elemento "ol", a ordenação tem significado. Como regra de ouro para determinar qual dos dois usar, tente mudar a posição dos itens na lista; caso a significação seja alterada, o elemento "ol" deve ser utilizado mas, se não houver mudança, você pode aplicar "ul".
